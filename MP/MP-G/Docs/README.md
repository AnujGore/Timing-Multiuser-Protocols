# MP-G+ Documentation

This folder stores the software and resources for the MP-G+ routing.

### Overview

Here’s a brief overview of the structure of the ``MP-G`` folder:

```
MP-G/
├── Scripts/
    ├── objects/
        ├── agent.py
        └── environment.py
    ├── config_files/
    ├── training.py
    └── utils/                          # Utility functions for data processing
        └── general_utils.py
├── Outputs/
    ├── action_files/
        └── (Folder to store the actions)
├── Docs/                               
    └── README.md                       
```
